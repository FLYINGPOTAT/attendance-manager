# やる気可視化 出席管理システム
出席管理システム（α版）<br>
出席状況に点数を付けて管理することで、その人のやる気を可視化することができます。

# Usage
### 0. HOME画面<br>
今月の出席状況とポイントの累積が表示されます。
![スクリーン ショット 2024-04-25 に 16 52 31 午後](https://github.com/FLYINGPOTAT/attendance-manager/assets/99582134/96d75c46-9983-4886-b66e-78c84bdd9861)

### 1. 部員登録
1-1. '部員管理'をクリック
<img width="1920" alt="image" src="https://github.com/FLYINGPOTAT/attendance-manager/assets/99582134/20950d09-c87d-44b0-8413-217e660a2367">

1-2. 遷移先のページで情報を入力
![スクリーン ショット 2024-04-25 に 17 11 12 午後](https://github.com/FLYINGPOTAT/attendance-manager/assets/99582134/08fd1b27-b9c9-44a9-a3c0-c03228783a77)

### 2. 出欠ポイントの設定
2-1. '出欠ポイント設定'をクリック
<img width="1920" alt="image" src="https://github.com/FLYINGPOTAT/attendance-manager/assets/99582134/45901039-b1fc-4173-9053-023de8829d84">

2-2. 出席状況に対応するポイントをセット
![スクリーン ショット 2024-04-25 に 17 15 34 午後](https://github.com/FLYINGPOTAT/attendance-manager/assets/99582134/64082d78-c28c-47b3-817f-546930b05983)

### 3. 出欠状況登録
3-1. '出席管理'をクリック
<img width="1920" alt="image" src="https://github.com/FLYINGPOTAT/attendance-manager/assets/99582134/1275a9a1-587c-4e08-93a6-6b466031110b">
3-2. 日付、名前、出欠を入力して送信<br>
名前や出欠は登録したものの中から選択する
![スクリーン ショット 2024-04-25 に 17 19 53 午後](https://github.com/FLYINGPOTAT/attendance-manager/assets/99582134/e70b9306-75c0-4acc-9f1c-8268afe1d142)

# Features
* 出席状況に点数を付けて管理することで、その人のやる気を可視化できる
* 自分でポイントの設定が可能
* 登録状況をDBに記憶
